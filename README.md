# Quiz-builder

Quiz-builder is the app for creating quizzes. The homepage (Quizzes Page) display the list of created quizzes and you can also add a new one by redirecting to the Create Page. When creating a quiz, it's possible to set the Answer type: true/false, text, or multiple choice, marking the correct answer. There is an opportunity to view the Quiz details in read-only mode. You can delete the Quiz as well.

# How to start

1. Clone the project from GitHub
2. Open the project in VScode
3. `cd /backend`
4. `npm install`
5. `npx prisma migrate dev --name init`
6. `npm run start:dev`
7. `../`
8. `cd /frontend`
9. `npm install`
10. `npm run dev`
